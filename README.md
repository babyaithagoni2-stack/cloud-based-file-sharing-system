# Cloud Based File Sharing System

A web application that allows users to upload, share and manage files securely.

## Technologies Used
- Frontend: HTML, CSS
- Backend: Python Flask
- Database: SQLite

## Features
- User Registration and Login
- File Upload and Download
- File Delete
- Secure Authentication

## How to Run
1. Install Python and Flask
2. Run `python app.py`
3. Open browser and go to `http://127.0.0.1:5000`
